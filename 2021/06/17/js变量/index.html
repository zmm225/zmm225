#变量

1.变量

变量是用来存储数据
变量是一个命名的存储空间
变量允许修改里面的值
变量在最后关闭文件的时候,会自动释放内存空间
变量是由:关键词(var 可以省略) 变量名
变量值,变量类型构成的
变量名自己定义:不能使用数字开头;后面的可以跟数字,字母下划线;不能使用关键词
变量名区分大小写
变量名的定义方式:
1) var demo = 123;
2) var demo
demo=123;
3) demo=123;
如果添加了var,则当前变量会预先加载

  1. 数据类型##JS的数据类型
    JS采用的是弱类型的方式,即不必声明数据类型,而是在使用或赋值时确定其数据类型

    ###7种数据类型(5类原始类型)前五个 通过typeof运算符判断数据类型

    string 字符串 拼接字符串用加号“+” 单双引号不能嵌套如若嵌套需要转义“\”

    number 数字 整型,浮点型,NaN(not a number)

    boolean 布尔 true和false表示判断的结果

                 表示假:数字0.0,数字0,字符串的空,null,undefined
             表示真:-10 '0000'  '0'  0.1 \0120

    undefined 未定义 定义了一个变量但没有为该变量赋值,使用一个并未赋值的变量,或者是使用了一个不存在的对象的属性时,JavaScript会返回undefined

    null 空 它不等于空字符串、“”、‘’、或者0 null与undefined的区别是,null表示一个变量被赋予了一个空值,而undefined则表示该变量尚未被赋值

    Array 数组 创建数组的三种方法:1. var 数组名:=new Array();2. var cars=new Array(“Audi”,”BMW”,”BYD”);3. var cars = [“Audi”,”BMW”,”BYD”];

    object 对象 面向对象 object js -> 对象 one.age one.play()

  2. 字符串,只要添加引号名,就是字符串 拼接符号:+
    字符串和字符串,字符串和变量,变量和变量
    单引号想要嵌套单引号,可以使用转义字符
    单引号和双引号不能识别变量

  3. 数字类型
    整型/浮点型[单精度浮点型,双精度浮点型]
    二进制(0b),八进制(0),十进制,十六进制(0x)
    128 => 10000000
    num.toFixed(保留位数)转换之后会变成字符串类型浮点数是一个近似值
    NaN(not a number),当程序由于某种原因计算错误的时候,将会产生一个无意义的数字NaN
    NaN与任何值都不相等,也包括本身
